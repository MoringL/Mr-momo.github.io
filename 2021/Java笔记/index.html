二、变量、标识符、保留字、变量

关键字

所有的关键字都是小写

保留字

Java现有版本暂未使用的，但后续版本可能作为关键字使用，自己应当避免使用。

如goto、const

标识符

  • 可含有26个字母、下划线、美元符号$

  • 大小写区分

  • 不可以使用关键字和保留字

  • 不能以数字作为开头（阿里巴巴Java开发手册中还强制不允许以下划线和美元符号开头结尾）

  • 不包含空格
